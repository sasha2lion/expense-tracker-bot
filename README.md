# Expense Tracker Bot (Simulation)

A simulated Telegram bot for tracking expenses.

## Features

- Add expenses with category  
  `/add coffee 5 food`
- List all expenses  
  `/list`
- Show total  
  `/total`
- Filter by category  
  `/category food`

## Example

/add coffee 5 food  
/add taxi 10 transport  
/list  
/total
